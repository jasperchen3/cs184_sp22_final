CS 184 Final Project

Ahmed Baqai, Dustin Luong, Ethan Chen, Jasper Chen

Title: Rendering Volumetric Scattering due to Airborne Particles

Milestone Status Report

In the project so far, we have read through several research papers that modeled behavior of light through media in a scene mathematically and have attempted to implement those models in code. We have worked starting from the ray tracer implemented in project 3-1, but are encountering difficulties in rendering the scene at the time of the milestone deadline. We will continue to work to implement a preliminary model of media such as fog in our scene, but have other ideas with respect to our final implementation.
